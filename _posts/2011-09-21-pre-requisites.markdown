<p>The following requirements need to be met in order for a successful installation</p>
<p><br> 1/ Backup your Boonex Dolphin files and MySQL database<br> 2/ Download <a href="http://www.joomla.org/">Joomla</a>. <br> 3/ Download <a href="http://www.jomsocial.com">JomSocial</a> extension<br> 4/ [Optional] Download <a href="http://www.azrul.com/products/jom-comment.html">JomComment</a> extension (user comments for content - not needed for blogs / forum)<br> 5/ [Optional] Download <a href="http://www.google.com.au/url?sa=t&amp;source=web&amp;cd=1&amp;ved=0CB0QFjAA&amp;url=http%3A%2F%2Fwww.kunena.org%2F&amp;rct=j&amp;q=kunena&amp;ei=i9d5Tsu8BqqciAeu37QS&amp;usg=AFQjCNHaxjb5-Oj60SDBz9KeIgHO1NY8dg&amp;sig2=HB2EvajXlGerY_hMsZlv8w&amp;cad=rja">Kunena</a> extension (user forums)<br> 6/ [Optional] Download <a href="http://www.google.com.au/url?sa=t&amp;source=web&amp;cd=1&amp;ved=0CB0QFjAA&amp;url=http%3A%2F%2Fwww.akeebabackup.com%2F&amp;rct=j&amp;q=akeeba&amp;ei=nNd5Tt3ONsOhiAfa3pD_Dw&amp;usg=AFQjCNGHoyFXTuQI_VTdIkbHTbTFhvMl4w&amp;sig2=uTeRwUs7jrc0VynngVMyKA&amp;cad=rja">Akeeba</a> extension (backup management)<br> 7/ [Optional] Download Stack Ideas <a href="http://stackideas.com/easyblog.html">EasyBlog</a> extension (blogs)</p>
<p>Please check the compatibility article to ensure that you use the correct versions.</p>
<p>We recommended that you consider installing the AKEEBA Backup extension to manage taking <br> backups of your site. This can also be automated by the use of a cron job.<br> <br> If you intend to migrate your forums you will need to install the KUNENA <br> extension. This is a mature forum that fully integrates with JomSocial and is <br> comparable in features and operation with phpBB or SMF<br> <br> The Stack Ideas EasyBlog extension is a very capable blog component. As with some of the other components, it is not free, but worth the small amount it costs.</p>
